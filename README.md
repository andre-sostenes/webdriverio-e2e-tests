# 🤖 Automação de Testes com WebdriverIO

## 📌 Objetivo

Projeto inicial de automação de testes E2E utilizando WebdriverIO para simular interações de usuário em uma aplicação web.

---

## 🧪 Funcionalidades testadas

* Acesso à aplicação
* Interação com elementos da página
* Execução de fluxo automatizado

---

## 🧰 Tecnologias utilizadas

* JavaScript
* WebdriverIO
* Node.js

---

## ⚙️ Como executar

1. Instalar dependências:
   npm install

2. Executar os testes:
   npx wdio run wdio.conf.js

---

## 🧠 Aprendizados

* Configuração do WebdriverIO
* Interação com elementos DOM
* Execução de testes automatizados E2E
